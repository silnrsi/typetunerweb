This package contains a TypeTuned version of the Lateef font.

This font is owned by SIL International and licensed under 
the Open Font License. See OFL.txt for more information.

The TypeTuner settings file used to generate the enclosed font
is included in the package as Lateef-feat_set_tuned.xml.
By inspecting this file with a text editor you can determine 
which alternate behaviors have been enabled.

For more information about this font, please visit
http://scripts.sil.org/ArabicFonts.
